These examples demonstrate how to correctly mark up HTML tables to ensure they are fully navigable and understandable by users relying on assistive technologies.

## Table 1: The Standard Simple Table

This is the standard, most robust way to build a table. 
* <a href="https://www.w3.org/WAI/WCAG21/Techniques/html/H39">WCAG Technique H39</a>: Uses a descriptive `caption`.
* <a href="https://www.w3.org/WAI/WCAG21/Techniques/html/H63">WCAG Technique H63</a>: Uses `scope="col"` and `scope="row"` to explicitly define header relationships.
* **Data Separation:** Every piece of data has its own semantic cell; no `<br>` tags are used for layout.

## Tables 2 & 3: Breaking Down Complex Tables

As recommended by accessibility guidelines, rather than creating a massive table with complex `colspan` and `rowspan` attributes, complex data should be broken up into simpler, highly readable tables. 

These tables also demonstrate <a href="https://www.w3.org/WAI/WCAG21/Techniques/general/G18">WCAG Technique G18</a>, using high-contrast zebra striping to aid users with cognitive or reading difficulties without sacrificing mathematical contrast requirements.

## Table 4: The Complex Data Table

While simplifying tables is preferred, if you must use a complex layout with merged cells, you must explicitly link the data to the headers.

* <a href="https://www.w3.org/WAI/WCAG21/Techniques/html/H43">WCAG Technique H43</a>: Uses the `id` attribute on header cells and the `headers` attribute on data cells to unambiguously associate them, even across nested columns.

<Tabs>
  <TabItem label="Rendered View">
    <table class="a11y-table">
        <caption>Employee Equipment Assignments</caption>
        <tr>
            <th rowspan="2" id="hdr-employee">Employee Name</th>
            <th colspan="2" id="hdr-equipment">Assigned Equipment</th>
        </tr>
        <tr>
            <th id="hdr-type" headers="hdr-equipment">Type</th>
            <th id="hdr-model" headers="hdr-equipment">Model</th>
        </tr>
        <tr>
            <th id="emp-john" headers="hdr-employee">John Doe</th>
            <td headers="emp-john hdr-equipment hdr-type">Laptop</td>
            <td headers="emp-john hdr-equipment hdr-model">Dell XPS 15</td>
        </tr>
        <tr>
            <th id="emp-jane" headers="hdr-employee">Jane Smith</th>
            <td headers="emp-jane hdr-equipment hdr-type">Tablet</td>
            <td headers="emp-jane hdr-equipment hdr-model">iPad Pro</td>
        </tr>
    </table>
  </TabItem>
