---
title: Page Regions
description: An introduction to HTML5 page regions and landmarks.
---

Pages with well-structured content are essential for many users. Screen reader users can skip to the main content directly and navigate to sections that are important to them. People with cognitive and learning disabilities can more easily find and prioritize content, and mobile web users often have access to a "reading" mode that relies on correctly marked-up main content.

HTML5 natively supports the following regions: Header (`<header>`), Footer (`<footer>`), Navigation (`<nav>`), Main content (`<main>`), and Complementary content (`<aside>`). 

Below is a complete HTML document exercising all of these regions with content appropriate for each container.

### Comprehensive Example

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>SpaceTeddy Inc. - Home</title>
</head>
<body>

    <header>
        <img src="logo.png" alt="SpaceTeddy Inc. Logo">
        <h1>SpaceTeddy Inc.</h1>
        <form role="search">
            <label for="site-search">Search the site:</label>
            <input type="search" id="site-search" name="q">
            <button type="submit" aria-label="Search">Search</button>
        </form>
    </header>

    <nav aria-label="Main Navigation">
        <ul>
            <li><a href="/">Home</a></li>
            <li><a href="/products">Products</a></li>
            <li><a href="/about">About Us</a></li>
            <li><a href="/contact">Contact</a></li>
        </ul>
    </nav>

    <main>
        <h2>Welcome to SpaceTeddy!</h2>
        <p>We create the galaxy's finest stuffed bears for aspiring astronauts.</p>
        
        <article>
            <h3>Featured Bear: The Apollo</h3>
            <p>Our best-selling bear comes with a tiny replica spacesuit.</p>
        </article>
    </main>

    <aside aria-labelledby="related-heading">
        <h2 id="related-heading">Related Links</h2>
        <ul>
            <li><a href="/history">History of Space Teddies</a></li>
            <li><a href="/materials">Our Synthetic Felt Paws</a></li>
        </ul>
    </aside>

    <footer>
        <p>&copy; 2026 SpaceTeddy Inc. All rights reserved.</p>
        <ul>
            <li><a href="/privacy">Privacy Policy</a></li>
            <li><a href="/terms">Terms of Service</a></li>
        </ul>
    </footer>

</body>
</html>
```

### WCAG Considerations
* [WCAG 1.3.1: Info and Relationships (Level A)](https://www.w3.org/WAI/WCAG22/Understanding/info-and-relationships.html): Using these native HTML5 sectioning elements provides semantic landmarks that convey the information structure programmatically.
* [WCAG 2.4.1: Bypass Blocks (Level A)](https://www.w3.org/WAI/WCAG22/Understanding/bypass-blocks.html): Properly defined regions (like `<main>`) allow users to bypass repetitive blocks of content (like navigation menus).
